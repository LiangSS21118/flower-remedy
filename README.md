# 新巴赫花精療法筆記

這個資料夾目前包含兩份原始 PDF 與整理後的 Markdown 筆記。

## 原始資料

- `前導課程-38朵巴赫花精介紹v12.pdf`：課程投影片，包含巴赫醫師理念、七大分類、急救花精與 38 朵花精介紹。
- `38朵花簡介加正面特質20260315.pdf`：一頁式總表，包含 38 朵花精的負面特質與正面頻率。

## 整理後筆記

- `巴赫花精整理筆記.md`：整合兩份 PDF 的重點，包含：
  - 課程核心觀念
  - 七大分類
  - 急救花精
  - 38 朵花精總表
  - 選花精時可以使用的判斷流程
- `巴赫花精整理筆記.html`：網頁版首頁，只顯示首頁重點花精。
- `core.html`：課程核心觀念。
- `categories.html`：七大分類。
- `rescue.html`：急救花精。
- `flowers.html`：38 朵花精查找、搜尋與分類篩選。
- `decision.html`：選花精的簡易判斷流程。
- `records.html`：自我紀錄區。
- `comparison.html`：容易混淆的花精比較。
- `review.html`：複習建議。
- `bach-flower-sync.json`：可綁定的固定同步 JSON 檔範本。

## 使用方式

建議先看 `巴赫花精整理筆記.md` 的「七大分類」，再用「38 朵花精總表」查找對應花精。

若要用網頁版，直接用瀏覽器開啟 `巴赫花精整理筆記.html`，再用上方導覽切換到其他獨立頁面。

首頁重點花精與自我紀錄會存在同一個瀏覽器本機。

## 固定 JSON 同步

若要讓首頁重點花精與自我紀錄自動同步到固定 JSON：

1. 用支援 File System Access API 的瀏覽器開啟 `巴赫花精整理筆記.html`，例如 Chrome 或 Edge。
2. 在首頁或 `records.html` 的「固定 JSON 同步」區塊按「綁定 / 建立 JSON」。
3. 選擇或建立 `bach-flower-sync.json`。
4. 之後新增、完成、刪除首頁重點或自我紀錄時，會自動覆寫同一個 JSON。

若換瀏覽器或清除網站資料，需要重新綁定一次 JSON 檔。
