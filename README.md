## 使用方式

建議先看 `03_整理筆記/巴赫花精整理筆記.md` 的「七大分類」，再用「38 朵花精總表」查找對應花精。

若要用網頁版，直接用瀏覽器開啟根目錄的 `index.html`，或直接開啟 `01_網站/巴赫花精整理筆記.html`，再用上方導覽切換到其他獨立頁面。

首頁重點花精與自我紀錄會存在同一個瀏覽器本機。

## 固定 JSON 同步

若要讓首頁重點花精與自我紀錄自動同步到固定 JSON：

1. 用支援 File System Access API 的瀏覽器開啟 `01_網站/巴赫花精整理筆記.html`，例如 Chrome 或 Edge。
2. 在首頁或 `records.html` 的「固定 JSON 同步」區塊按「綁定 / 建立 JSON」。
3. 選擇或建立 `04_個人同步/bach-flower-sync.json`。
4. 之後新增、完成、刪除首頁重點或自我紀錄時，會自動覆寫同一個 JSON。

若換瀏覽器或清除網站資料，需要重新綁定一次 JSON 檔。
