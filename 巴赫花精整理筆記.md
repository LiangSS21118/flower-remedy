# 巴赫花精整理筆記

> 用途：整理成方便複習、查找與自我觀察的筆記。

## 一、課程核心觀念

新巴赫花精療法課程中提到的架構包含：38 朵花精、12 個花軌、243 個花精皮膚反應區、4 種療癒層次。

巴赫花精的重點不是直接針對疾病名稱，而是觀察「人目前處在什麼情緒狀態」。課程中反覆出現的核心邏輯是：

1. 花精代表某種正向美德或正面頻率。
2. 選花精時，要先辨認目前主要的負面情緒、反應模式或卡住的狀態。
3. 花精的目標是協助負面情緒回到較平衡的狀態。
4. 重點是「治療生病的人」，而不是只看疾病本身。
5. 疾病被視為提醒人看見人格與靈魂不和諧的訊號，恢復健康需要找出錯誤並開展相對應的美德。

簡單說：先看情緒與人格狀態，再對應需要的花精。

## 二、七大分類

| 分類 | 主要狀態 | 對應花精 |
|---|---|---|
| 1. 恐懼 | 害怕、恐慌、擔憂自己或他人 | 白楊、櫻桃李、溝酸漿、紅栗花、岩薔薇 |
| 2. 不確定 | 猶豫、懷疑、缺乏方向或信心 | 水蕨、龍膽、金雀花、角樹、線球草、野燕麥 |
| 3. 對現況不感興趣 | 不在當下、沉在過去或未來、疲憊、重複犯錯 | 栗樹芽苞、鐵線蓮、忍冬、歐白芥、橄欖、白栗花、野薔薇 |
| 4. 孤單 | 孤獨、急躁、不易與人連結 | 石楠、鳳仙花、水堇 |
| 5. 易受外界影響 | 壓抑、討好、容易被他人左右或情緒牽動 | 龍芽草、矢車菊、冬青、胡桃 |
| 6. 沮喪、絕望 | 自責、壓力、創傷、絕望、失去力量 | 酸蘋果、榆樹、落葉松、橡樹、松樹、伯利恆之星、甜栗花、楊柳 |
| 7. 過度牽掛別人的安康 | 批判、控制、僵化、過度熱忱、壓迫他人 | 櫸木、菊苣、岩水、馬鞭草、葡萄藤 |

## 三、急救花精 Rescue Remedy

急救花精是五花組合，課程中整理為適合「緊急心理狀態」或「生理傷害」時參考，例如驚嚇、打擊、恐懼、噩耗、意外、生病、昏迷等。

| 花精 | 對應狀態 |
|---|---|
| 伯利恆之星 Star of Bethlehem | 驚嚇、心靈受創、需要安慰 |
| 岩薔薇 Rock Rose | 恐慌、極度害怕、動彈不得 |
| 櫻桃李 Cherry Plum | 快抓狂、精神壓力大、害怕失控 |
| 鐵線蓮 Clematis | 昏沉、失去意識、不在當下 |
| 鳳仙花 Impatiens | 急躁、緊繃、疼痛時的焦躁 |

注意事項：急救花精不能在手術麻醉前使用。

## 五、選花精的簡易判斷流程

可以用下面的順序來整理自己的狀態：

1. 先問：我現在最明顯的是哪一類情緒？
   - 害怕？不確定？不在當下？孤單？容易被影響？絕望？過度關心別人？
2. 再看：這個狀態比較像哪一朵花的描述？
   - 例如「怕具體事情」偏向溝酸漿，「莫名害怕」偏向白楊。
3. 最後看：我想要回到哪個正面方向？
   - 例如「信任自己的判斷」、「選擇希望」、「安穩思緒」。

## 七、容易混淆的花精比較

| 容易混淆 | 差異 |
|---|---|
| 白楊 vs 溝酸漿 | 白楊是說不出原因的恐懼；溝酸漿是可以明確說出怕什麼。 |
| 龍膽 vs 金雀花 | 龍膽是遇到阻礙後悲觀；金雀花是長期失望後幾乎不抱希望。 |
| 角樹 vs 橄欖 | 角樹偏心智疲憊、開始前覺得累；橄欖偏身心都耗竭，真的沒力。 |
| 鐵線蓮 vs 忍冬 | 鐵線蓮活在未來或幻想；忍冬停留在過去。 |
| 橡樹 vs 榆樹 | 橡樹是長期硬撐、扛責任；榆樹是有能力但某段時間覺得負荷過大。 |
| 菊苣 vs 紅栗花 | 菊苣是過度關懷並帶控制；紅栗花是過度擔心所愛的人出事。 |
| 馬鞭草 vs 葡萄藤 | 馬鞭草是熱忱地想說服別人；葡萄藤是強勢地要求別人照自己的方式做。 |

## 八、複習建議

- 若要快速背誦：先背七大分類，再背每類有哪些花精。
- 若要自我覺察：每天選一個最明顯的情緒狀態，對照總表。
- 若要做課程筆記：可以在每朵花精後面補上自己的例子，例如「我什麼時候會像這朵花的狀態」。
